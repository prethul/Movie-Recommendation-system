# 🎬 Pre-Flix Movie Recommendation System

Pre-Flix is a simple Machine Learning-based Movie Recommendation System built with **Python** and **Streamlit**. The application uses a **content-based recommendation algorithm** to suggest **5 similar movies** based on the movie selected by the user.

This project is designed as a beginner-friendly machine learning application that demonstrates how recommendation systems work and how they can be deployed using Streamlit.

---

## 🚀 Features

- Search for your favorite movie
- Get 5 recommended similar movies instantly
- Simple and interactive Streamlit interface
- Content-Based Filtering recommendation model
- Fast prediction using preprocessed movie similarity data

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Pickle

## ⚙️ How It Works

1. Select or search for a movie.
2. The trained recommendation model finds similar movies using content-based filtering.
3. The system predicts and displays **5 recommended movies** that are most similar to the selected movie.

> **Note:** This is a basic movie recommendation model created for learning and demonstration purposes.

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Pre-Flix-Movie-Recommendation-System.git
```

Go to the project directory:

```bash
cd Pre-Flix-Movie-Recommendation-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app.py
```
